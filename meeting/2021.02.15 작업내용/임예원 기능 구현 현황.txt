임예원 작업
- 노하우게시판 파트

--------------------------------------------------------------------------

기능(페이지별)
1. board_KnowHowListPage
    - 게시판 목록 나열
    - 새글 작성하기누르면 insertPage 이동
    - 카테고리 태그 필터링기능
    **검색 기능(보류)
       로그인 했을때만 새글 작성할수 있는 기능 필요
       추천글 내가 적은 번호 기입 기능 구현 필요
       글 목록 페이징 기능

2. board_knowHowInsertPage
    - 게시글 작성하기
    - 카테고리 작업 기능필요
    - 글 작성기능
    - 작성 완료 버튼 클릭시 viewPage 이동
    - 목록으로 돌아가기 버튼 클릭시 listPage이동
   
    
3. board_knowHowViewPage
    - 조회수기능
    - 작성자(닉네임 기능 필요)
    - 제목,내용 보기 기능
    - 수정버튼 클릭시 updatePage로 이동
    - 삭제버튼 클릭시 글 삭제기능
    - 글 목록 클릭시 listPage이동
    ** 게시글 댓글 기능 추가
    
4. board_knowHowUpdatePage
   - 제목 내용 수정완료후 버튼 클릭하면 수정정보 포함 viewPage로 이동
   - 취소 버튼 클릭시 상태 그대로 viewPage로 이동
